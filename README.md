 🏡 Luxury Housing Sales Analysis

📌 Project Overview

This project analyzes luxury housing data in Bangalore to uncover trends in sales, revenue, and customer conversion across different micro-markets.

The objective is to generate actionable insights that can help real estate developers and stakeholders make data-driven decisions.

---

 📊 Key Dashboard Features

 🔹 1. Quarterly Sales Trend

* Tracks total sales across quarters
* Helps identify seasonal patterns and performance fluctuations

 🔹 2. Amenity Score vs Conversion Rate

* Shows relationship between property amenities and customer conversion
* Bubble size represents total sales

 🔹 3. Sales by Configuration

* Compares performance of different property types (3BHK, 4BHK, 5BHK+)
* Identifies high-demand configurations

 🔹 4. Sales Distribution by Micro-Market (Map)

* Visualizes geographic concentration of sales
* Highlights high-performing regions

🔹 5. Key Insight Section

* Summarizes business insights directly from the dashboard
* Adds real-world decision-making value

## 💡 Key Insights

* Higher **amenity scores** generally lead to improved **conversion rates**
* Some **micro-markets outperform others**, indicating location advantage
* Sales are heavily **clustered in key urban zones**
* Larger configurations (5BHK+) contribute significantly to total revenue
